# ShoppingMall
2024.01~2024.03 / 1인 개인 프로젝트

# Introduce
로그인과 회원가입, 상품 등록, 구매와 결제, 후기, 장바구니와 같은 기능을 갖춘 쇼핑몰 웹 서비스를 개발했습니다. 
DB사이의 연관관계가 여럿 존재하기에 이를 고려하여 ERD를 설계했습니다. 
또 결제 부분에 대해서는 포트원을 통하여 결제요청 기능을 만들었으며, 결제 시 카카오페이로 테스트 결제가 되도록 개발하였습니다.
